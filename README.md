# Digital Library Management System

A Java Swing-based desktop application for managing books, issuing/returning books, and viewing records in a library.

## Features
- Add, update, delete, and search books
- Issue and return books
- View all books in a table
- Connects to a MySQL database via JDBC

## Requirements
- Java JDK 8 or newer
- MySQL database
- mysql-connector jar included in the project folder

## Run
Compile and run the project with Java after setting up the database connection details in DBConnection.java.
